Useful Color Palette Generators Online

Color Palette Generators​

Colormind​

http://colormind.io/

Pros:

  • A random color palette generator for creating color schemes randomly
  • Generate color scheme automatically using deep learning
  • Generate beautiful intermediary color shades
  • Allow you to lock more than one color each time

Cons:

  • Not able to download the color scheme generated

Coolors​

https://coolors.co/generate

Pros:

  • Drag and drop interface
  • An excellent color palette generator from hex
  • Choose different color picker mode, including HEX, HSB, RGB
  • A random color palette generator
  • Search images from Unsplash and extract colors from them

Cons:

  • Colors extracted from image look different from the actual one

MyColor​

https://mycolor.space/

Pros:

  • Simple and easy to use, able to generate related palettes in seconds
  • Has CSS color gradient generator

Cons:

Muzli​

https://colors.muz.li/

Pros:

  • A random color palette generator
  • Intuitive interface
  • Color palette generation from image
  • Has search engine

Cons:

  • Download in SVG format only
  • Generate 5 color palettes for free

Canva Color​

https://www.canva.com/colors/color-palettes/

Pros:

  • An effective color palette generator from image
  • Stunning color palettes to choose from
  • A nice brand color palette generator

Cons:

  • No color wheel

Scheme Color​

https://www.schemecolor.com/

Pros:

  • A random color palette generator
  • Huge list of color schemes category
  • Stunning color palettes to choose from

Paletton​

https://paletton.com/

Pros:

  • Color wheel is available
  • A random color palette generator
  • Highly customizable
  • Able to Preview as webpages

Cons:

  • Maybe too technical for normal user
